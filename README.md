# Calculadora de Descuentos Gratis

Aplicación web gratuita para **calcular descuentos** y el precio final de tus compras.

## ✨ Características

- Descuentos predefinidos: 10%, 25%, 50%, 70%.
- Porcentaje **personalizado**.
- Soporta **descuentos encadenados** (ej. rebaja + cupón).
- Muestra ahorro total y % efectivo.
- Cálculo en tiempo real.

## 🚀 Uso

1. Ingresa el **precio original**.
2. Selecciona o escribe el **descuento**.
3. Opcionalmente agrega un **descuento adicional**.
